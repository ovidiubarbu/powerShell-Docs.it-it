## <a name="listing-all-the-files-and-folders-within-a-folder"></a><span data-ttu-id="e06a8-106">Elenco di tutti i file e le cartelle all'interno di una cartella</span><span class="sxs-lookup"><span data-stu-id="e06a8-106">Listing All the Files and Folders Within a Folder</span></span>

<span data-ttu-id="e06a8-107">È possibile usare **Get-ChildItem** per ottenere tutti gli elementi direttamente all'interno di una cartella.</span><span class="sxs-lookup"><span data-stu-id="e06a8-107">You can get all items directly within a folder by using **Get-ChildItem**.</span></span> <span data-ttu-id="e06a8-108">Aggiungere il parametro facoltativo **Force** per visualizzare elementi nascosti o di sistema.</span><span class="sxs-lookup"><span data-stu-id="e06a8-108">Add the optional **Force** parameter to display hidden or system items.</span></span> <span data-ttu-id="e06a8-109">Ad esempio, questo comando visualizza il contenuto diretto dell'unità C di Windows PowerShell, che corrisponde all'unità fisica di Windows C:</span><span class="sxs-lookup"><span data-stu-id="e06a8-109">For example, this command displays the direct contents of Windows PowerShell Drive C (which is the same as the Windows physical drive C):</span></span>

```

<span data-ttu-id="e06a8-110">Il comando consente di elencare solo gli elementi contenuti direttamente, in modo molto simile al comando **DIR** di Cmd.exe o a **ls** in una shell UNIX.</span><span class="sxs-lookup"><span data-stu-id="e06a8-110">The command lists only the directly contained items, much like using Cmd.exe's **DIR** command or **ls** in a UNIX shell.</span></span> <span data-ttu-id="e06a8-111">Per visualizzare gli elementi contenuti, è necessario specificare anche il parametro **-Recurse**.</span><span class="sxs-lookup"><span data-stu-id="e06a8-111">In order to show contained items, you need to specify the **-Recurse** parameter as well.</span></span> <span data-ttu-id="e06a8-112">(L'esecuzione di questo comando può richiedere tempi estremamente lunghi.) ```

<span data-ttu-id="e06a8-113">**Get-ChildItem** consente di filtrare gli elementi tramite i parametri **Path**, **Filter**, **Include** ed **Exclude**, ma questi parametri sono solitamente basati solo sul nome.</span><span class="sxs-lookup"><span data-stu-id="e06a8-113">**Get-ChildItem** can filter items with its **Path**, **Filter**, **Include**, and **Exclude** parameters, but those are typically based only on name.</span></span> <span data-ttu-id="e06a8-114">È possibile eseguire operazioni di filtraggio complesse in base ad altre proprietà degli elementi tramite il cmdlet **Where-Object**.</span><span class="sxs-lookup"><span data-stu-id="e06a8-114">You can perform complex filtering based on other properties of items by using **Where-Object**.</span></span>

<span data-ttu-id="e06a8-115">Il comando seguente consente di trovare tutti i file eseguibili all'interno della cartella Programmi modificati per l'ultima volta dopo il 1 ottobre 2005 e che non hanno dimensioni minori di 1 o maggiori di 10 MB:</span><span class="sxs-lookup"><span data-stu-id="e06a8-115">The following command finds all executables within the Program Files folder that were last modified after October 1, 2005 and which are neither smaller than 1 megabyte nor larger than 10 megabytes:</span></span>

```
